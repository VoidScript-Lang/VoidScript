hello.void 
[Import modules]
Void.import VGame
Void.import Void.math
Void.import Void.rand

[Define variables]
name : str = "VoidScript"
score : int = 0

[Create a window]
Void.window.create: 800, 600, name

[Draw shapes]
Void.window.square: 80, 80, "player", 100, 200

[Random bonus]
Void.rand.int: 10, 50
bonus : int = __last__
Void.say: bonus
Why VoidScript

⚡
Lightning Fast

Compiles and runs instantly. No heavy setup, no waiting — just write your .void file and go.

🎮
Game Ready

VGame (Pygame) built-in. Create windows, draw shapes, handle buttons and images without setup.

🔊
Audio Support

Play, pause, loop, and control volume of sound effects and background music with one-liners.

📦
20+ Modules

Math, randomness, networking, databases, images, hashing, CSV, zip, and more. All with Void syntax.

🧩
Clean Control Flow

if, elif, else, while, do-while, repeat, repeat.over, repeat.range — all readable, all consistent.

🧪
Built-in Testing

Void.test gives you assertions, type checks, and pass/fail reporting right in your script.
